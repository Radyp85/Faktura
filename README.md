# Fakturační Systém

React + TypeScript + Vite aplikace pro generování faktur pro neplátce DPH.

## Funkce
- Správa klientů (CRUD)
- Generátor faktur s automatickým číslováním
- Tisková sestava A4 s QR platbou (SPAYD standard)
- Podpis z obrázku
- Historie vystavených faktur
- LocalStorage persistence

## Spuštění

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
